# Equine Acres

A text-based horse farm management simulation game built with Next.js and Prisma.

## About

Equine Acres is a free and open-source farming simulation where you manage horses, crops, and livestock. Build your farm from humble beginnings into a thriving equestrian estate.

### Features

- **Horse Breeding & Training** - Raise horses with AI-generated personalities and bios
- **Dynamic Crop System** - Plant and harvest hay, corn, and cotton with weather effects
- **Livestock Management** - Manage chickens, cows, goats, and pigs
- **Achievement System** - Unlock achievements as you progress through the game
- **Daily Chores** - Clean stables, repair fences, water crops, and feed animals
- **Random Events** - Experience unexpected farm events that affect your gameplay
- **Economic Simulation** - Buy low, sell high with fluctuating market prices

## Tech Stack

- **Frontend**: Next.js 15 (App Router), React 19, TypeScript
- **Database**: PostgreSQL (Neon serverless) with Prisma ORM
- **Styling**: Tailwind CSS
- **AI Integration**: Google Gemini AI (optional, for horse names and bios)
- **Icons**: Lucide React

## Architecture

This game uses a modern full-stack architecture:
- Server Actions for database mutations
- Prisma for type-safe database access
- Client-side hooks for game state management
- Optimistic UI updates for responsive gameplay
